Siswati-English Translator

Welcome to Siswati-to-English Translator

Our model was trained using Python v3.7 with the JoeyNMT and PyTorch libraries. Please note that this is for educational purposes only, the model is very basic since we had limited data for training. For similar projects translating African languages to English, check out this repository, this model was heavily built upon it.

Note: The Siswati-to-English model was mostly trained on religious content (the Siswati bible, JW300 data etc.) since it is the only Siswati content readily available in the public domain, therefore religious bias is expected, such that the model can only translate in the religious context.
